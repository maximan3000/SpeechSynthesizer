Идея : Контролируем число запусков следующим образом:

	1) Берем дату создания файла base.dll
	2) Кодируем её в соответствии с таблицей encryption_table
	3) Закодированную строку по частям засовываем в 5 разных файлов
	4) При запуске воссоединяем строку из этих файлов и перекодируем её в дату
	5) По дате создания определяем файл, один из тех что хранил часть строки
	6) По номеру файла в массиве определяем число запусков и производим нужные действия
	7) Берем следующий элемент в массиве файлов и узнаем его дату создания
	8) Кодируем её в соответствии с таблицей encryption_table
	9) Закодированную строку по частям засовываем в 5 разных файлов

Ключ для сброса попыток: maks
